SEND DATA THROUGH SERIAL PORT

Install pyserail

Use pip to install pyserail module

pip install pyserial

Open serail port

First, import pyserial module

import serial

Second, open your serial port. If you don't know which port you are using, you can do it like this.

def open_serialport():
    PORT_NUM = 1
    while True:
        SERIAL_PORT = 'COM%d' % PORT_NUM
        try:
            ser = serial.Serial(port=SERIAL_PORT,baudrate=1048576,parity='N',bytesize=8,stopbits=1,timeout=0)
            print "OPEN SERIAL PORT ON %s" % SERIAL_PORT
            return ser
        except Exception,e:
            PORT_NUM = PORT_NUM + 1
